2. Specificity

Specificity menerangkan level kekuatan css. Selain berdasarkan urutan di file CSS, ada yang urutannya begini:

  1. Element (body, h1, p, dll) = 1
  2. Class (.) = 10
  3. Id (#) = 100
  4. Inline (dalem html) = 1000

Keterangannya, semakin besar maka akan lebih didahulukan untuk muncul
